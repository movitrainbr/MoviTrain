# MoviTrain

MoviTrain é uma plataforma para conectar pessoas com objetivos de treino semelhantes, criando consistência através da comunidade.

## Funcionalidades

- Encontrar parceiros de treino
- Agendar sessões
- Sistema de progresso
- Comunidade fitness
- Interface responsiva

## Tecnologias

- HTML5
- CSS3
- JavaScript
- Particles.js
